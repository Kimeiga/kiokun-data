# Demo Script - Kanji Component Poker
# This script demonstrates the core game mechanics

# Show initial status
status

# Try to find and play a valid 2-character word
suggest

# Combine two components to make a more complex character
combine 1 2

# Show status after combining
status

# Try to play a word with our new character
play 1 3

# Show updated status
status

# Dissolve a character back into components
dissolve 2

# Discard some cards we don't need
discard 1 2

# Show final status
status

# Try to play another word
play 3 4

# Continue playing to try to beat the blind
play 1 2

play 3 4 5

play 1 2 3
